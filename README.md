# Gerador de QR Code

Projeto simples que contém um gerador de QR Code em um único arquivo HTML.

Como usar:

- Abra o arquivo `gerador qr code.html` no navegador.
- Preencha o texto/URL e gere o QR Code.

Arquivo principal:
- `gerador qr code.html`
